# Морський бій (Java Desktop)
Проста десктопна реалізація гри «Морський бій» з графічним інтерфейсом на Swing. Після запуску відкривається стартове меню з вибором режиму, а під час матчу відкривається окреме вікно з двома полями: зліва ваше розташування кораблів, справа — сітка супротивника для стрільби. Кнопка «Нова гра» перемішує флот і перезапускає матч.

## Збірка та запуск
Потрібна встановлена JDK (17 або новіша). З кореня репозиторію виконайте:

```bash
# Компіляція у директорію out/
javac -d out $(find src -name "*.java")

# Запуск гри
java -cp out battleship.BattleshipGame
```

### Запуск Java-версії (Swing)
Після запуску `battleship.BattleshipGame` відкривається меню з пунктами:

- «Нова гра проти комп'ютера»
- «Нова локальна гра для двох гравців»
- «Завантажити збереження»
- «Створити онлайн-гру»
- «Підключитися до онлайн-гри»
- «Змінити мову»
- «Скинути статистику»
- «Вихід»

Обравши гру проти комп'ютера, ви переходите до поля бою, де можна почати стріляти по клітинках противника. 
## Правила гри
- Використовується класичний набір кораблів: 1×4, 2×3, 3×2 та 4×1 палуби.
- Кораблі генеруються випадково і не торкаються одне одного по діагоналі та по сторонах.
- Натискайте по клітинках праворуч, щоб стріляти. Комп’ютер відповідає своїм ходом автоматично.
- Перемагає той, хто першим потопить усі ворожі кораблі.
