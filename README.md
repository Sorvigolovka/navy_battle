# Морський бій (Desktop)

Це проста десктопна реалізація гри «Морський бій» з графічним інтерфейсом на `tkinter`.
Архітектура організована у стилі MVC: `game_model.py` зберігає стан полів та кораблів, `game_controller.py` керує ходами гравців, а `game_view.py` відповідає за інтерфейс і взаємодію користувача.

## Запуск

```bash
python main.py
```

Інтерфейс відкриється у новому вікні: зліва показано ваш флот, справа — поле супротивника для стрільби. Кнопка «Нова гра» перезапускає матч з новою випадковою розстановкою кораблів.
